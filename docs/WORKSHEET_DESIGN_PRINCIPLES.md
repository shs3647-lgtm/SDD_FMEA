# 워크시트 설계 원칙

**작성일**: 2026-01-14  
**적용 범위**: 모든 워크시트 화면 (FMEA, CP, PFD 등)

## 핵심 설계 원칙

### 1. 메뉴바 크기 및 고정

#### 메뉴바 크기
- **기준**: 브라우저 크기에 맞춰 개발
- **기본 크기**: 1440px (표준 화면 기준)
- **반응형**: viewport 크기에 따라 조정 가능
- **원칙**: 메뉴바는 브라우저 크기에 맞춰서 개발한다 (1440px 기준)

#### 메뉴바 고정
- **위치**: 항상 고정 (position: fixed 또는 sticky)
- **스크롤 영향 없음**: 가로/세로 스크롤 시에도 움직이지 않음
- **z-index**: 다른 요소 위에 항상 표시
- **원칙**: 메뉴바는 항상 고정되어 가로세로 스크롤에도 움직이지 않는다

**구현 방법**:
```css
/* 메뉴바 컨테이너 */
.menu-bar {
  position: fixed;
  top: 0;
  left: 53px; /* 사이드바 너비 + 간격 */
  width: 1440px; /* 기본 크기 */
  z-index: 99;
}
```

### 2. 스크롤바 정책

#### 좌우 스크롤바 (가로 스크롤)
- **원칙**: 컨테이너에 **1개만** 사용
- **위치**: 메인 워크시트 컨테이너
- **중첩 스크롤바 금지**: 내부 테이블이나 다른 요소에 추가 스크롤바 생성 금지
- **핵심 원칙**: 좌우 스크롤바는 컨테이너에 1개만 사용한다

**구현 방법**:
```css
/* 메인 워크시트 컨테이너 */
.worksheet-container {
  overflow-x: auto; /* 가로 스크롤 1개만 */
  overflow-y: hidden; /* 세로 스크롤은 워크시트에만 */
}
```

#### 세로 스크롤바 (세로 스크롤)
- **원칙**: 워크시트 영역만 조정
- **위치**: 워크시트 테이블 컨테이너
- **메뉴바/헤더 제외**: 메뉴바와 헤더는 스크롤에 포함되지 않음
- **핵심 원칙**: 세로 스크롤바는 워크시트만 조정하게 만든다

**구현 방법**:
```css
/* 워크시트 테이블 컨테이너 */
.worksheet-table-container {
  overflow-x: hidden; /* 가로 스크롤은 메인 컨테이너에서 */
  overflow-y: auto; /* 세로 스크롤만 여기서 */
}
```

### 3. 레이아웃 구조

```
┌─────────────────────────────────────┐
│ 사이드바 (고정, 48px)                │
├─────────────────────────────────────┤
│ 메뉴바 (고정, 1440px, 스크롤 없음)   │
├─────────────────────────────────────┤
│ 헤더 영역 (고정, 스크롤 없음)        │
├─────────────────────────────────────┤
│ 워크시트 컨테이너 (가로 스크롤 1개) │
│ ┌─────────────────────────────────┐ │
│ │ 워크시트 테이블 (세로 스크롤)    │ │
│ │                                 │ │
│ │ [세로 스크롤바]                 │ │
│ └─────────────────────────────────┘ │
│ [가로 스크롤바]                      │
└─────────────────────────────────────┘
```

## 구현 가이드

### 메뉴바 구현 예시

```tsx
// 메뉴바 컴포넌트
<div className="fixed top-0 left-[53px] w-[1440px] z-[99]">
  {/* 메뉴바 내용 */}
</div>

// 메인 콘텐츠 영역 (메뉴바 아래)
<div className="mt-[메뉴바높이]">
  {/* 콘텐츠 */}
</div>
```

### 스크롤 컨테이너 구조

```tsx
// 메인 워크시트 컨테이너 (가로 스크롤 1개만)
<div className="overflow-x-auto overflow-y-hidden">
  {/* 워크시트 테이블 컨테이너 (세로 스크롤만) */}
  <div className="overflow-x-hidden overflow-y-auto">
    <table>
      {/* 워크시트 테이블 */}
    </table>
  </div>
</div>
```
